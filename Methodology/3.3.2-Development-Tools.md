# 3.3.2 Development Tools

The current LYDO Connect implementation uses the following tools:

| Category | Tools |
|---|---|
| Frontend framework | React, Vite, TypeScript |
| Styling and UI | Tailwind CSS, shadcn/ui, Radix UI, Lucide React |
| Routing and state | React Router, TanStack Query, React context/state |
| Forms and validation | React Hook Form, Zod, resolver helpers |
| Backend and storage | Supabase, PostgreSQL, Supabase Storage |
| Testing and linting | Vitest, ESLint |
| Version control | Git and GitHub |
| Deployment | Vercel or a compatible static hosting platform |

These tools support the current app workflow, including authenticated access, policy acceptance, document upload and preview, budget request CRUD, liquidation file upload, news release preview, transparency posting, required document type management, notifications, and admin review screens.
