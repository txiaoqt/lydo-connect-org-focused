# 3.3.2 Development Tools

The current LYDO Connect implementation uses the following tools:

| Category | Tools |
|---|---|
| Frontend framework | React, Vite, TypeScript |
| Styling | Tailwind CSS, shadcn/ui |
| Routing and state | React Router, TanStack Query, custom React context/state |
| Icons and UI helpers | Lucide React, utility helpers |
| Backend and storage | Supabase, PostgreSQL, Supabase Storage |
| Validation and testing | Vitest, ESLint |
| Version control | Git and GitHub |
| Deployment | Vercel or a compatible static hosting platform |

These tools support the current app workflow, including authenticated access, document upload and preview, budget request CRUD, liquidation file upload, news release preview, transparency posting, and admin review screens.
