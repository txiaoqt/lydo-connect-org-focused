# Statistical Treatment

This section presents the statistical methods used to interpret and summarize the data gathered for **LYDO Connect: A Progressive Web Application for Information, Participation, and Transparency for Local Youth Development Offices**. In alignment with the study manuscript and the implemented site workflows, the evaluation used simple descriptive statistics, specifically frequency, percentage, and weighted mean, for survey-based assessment.

Frequency and percentage were used to describe respondent profiles and to show how many respondents selected each response option for every item. These measures were applied to summarize user and evaluator responses across the system's major workflow areas, including information access, participation/registration, transparency viewing, service support, and administrative operations.

The weighted mean was used to determine the level of agreement for each indicator in the 5-point Likert-scale instruments. The resulting mean scores were interpreted using verbal descriptions to determine the system's acceptability and performance. This treatment supports the study's evaluation objectives for usability, performance efficiency, reliability, security, and overall service quality, including ISO/IEC 25010-aligned expert/admin criteria.

The following formulas were applied:

```text
Percentage (%) = (f / N) x 100
```

Where:
- `f` = frequency of responses
- `N` = total number of respondents

```text
Weighted Mean (WM) = (SUM(f * w)) / N
```

Where:
- `f` = frequency of responses per scale value
- `w` = weight of each scale value
- `N` = total number of responses

For Likert-scale items, the response scale used was:
- 5 = Strongly Agree
- 4 = Agree
- 3 = Neutral
- 2 = Disagree
- 1 = Strongly Disagree

The weighted mean results were interpreted using this guide:

| Weighted Mean Range | Verbal Interpretation |
|---|---|
| 4.21 - 5.00 | Strongly Agree / Highly Acceptable |
| 3.41 - 4.20 | Agree / Acceptable |
| 2.61 - 3.40 | Neutral / Moderately Acceptable |
| 1.81 - 2.60 | Disagree / Needs Improvement |
| 1.00 - 1.80 | Strongly Disagree / Not Acceptable |

For interpretation, item-level weighted means were computed first, then summarized per evaluation dimension (for example: Information Quality, System Quality, Service Quality, and ISO/IEC 25010 quality characteristics used in the expert/admin instrument). Overall results were then compared against the verbal interpretation ranges to determine whether outcomes were acceptable or needed improvement.

In summary, the statistical treatment focused on numerical processing and summarization of evaluation data using frequency, percentage, and weighted mean, while broader data analysis integrated these quantitative results with qualitative feedback to support conclusions and recommendations for LYDO Connect.
