# Statistical Treatment

This section presents the descriptive statistics used to summarize evaluation results for LYDO Connect.

## Descriptive Measures

| Statistic | Purpose |
|---|---|
| Frequency | Summarize respondent counts and answer distributions |
| Percentage | Show the proportion of each response category |
| Weighted mean | Determine the level of acceptability for each criterion |
| Standard deviation | Measure the consistency of responses |

## Interpretation Guide

| Weighted Mean Range | Interpretation |
|---|---|
| 4.21 - 5.00 | Excellent / Highly Acceptable |
| 3.41 - 4.20 | Very Good / Acceptable |
| 2.61 - 3.40 | Good / Moderately Acceptable |
| 1.81 - 2.60 | Fair / Needs Improvement |
| 1.00 - 1.80 | Poor / Not Acceptable |

## Scope

The statistical treatment applies to the current app workflow only: authentication, organization profile, document submission, budget request CRUD, liquidation reporting, news releases, transparency posts, templates, notifications, and admin review.
