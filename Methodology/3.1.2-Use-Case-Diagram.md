## Notes for Manuscript Use

- These are actual diagram files, not code blocks.
- The files are stored as SVG so they stay sharp when resized in Word.
- The public diagram now includes account access, programs, events, organizations, profile, transparency, financial disclosure, barangay map, citizen desk, ticket tracking, service advisories, and registration sync support.
- The admin diagram now includes dashboard, records management, barangay and resident management, transparency documents, transparency board, financial DSS, registrations and sync monitoring, citizen desk administration, users and roles, and audit logs.
- If needed, you can insert the SVG files directly from the `Methodology/diagrams` folder.
