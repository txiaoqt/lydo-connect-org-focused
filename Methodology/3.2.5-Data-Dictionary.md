# 3.2.5 Data Dictionary

The data dictionary defines the main tables and fields used by the current LYDO Connect workflow and implemented schema. It follows the organization-focused implementation reflected in the current workflows: account access, policy acceptance, organization registration, compliance submission, budget and liquidation processing, YPOP validation, YORP-related organization registry fields, content publishing, notifications, and activity logs. Tables that exist only for legacy compatibility outside the present workflow narrative are not expanded in this section.
