# 3.2.5 Data Dictionary

The data dictionary defines the major tables, fields, data types, and meanings used by LYDO Connect. This version is based on the implemented Supabase schema and the supporting migrations used by the project.

## Notes

- `auth.users` is managed by Supabase Authentication and is referenced by several tables.
- `admin_accounts` is not included in the core dictionary because it is an optional helper table for predefined admin mode rather than a primary business entity of the study.
- Enum values are listed first so that field meanings remain consistent across tables.

## Enumerated Data Types

| Enum Name | Allowed Values | Purpose |
|---|---|---|
| `app_role_code` | `admin`, `staff`, `sk`, `youth` | Defines application roles |
| `program_status` | `draft`, `published`, `archived` | Defines program visibility state |
| `event_status` | `draft`, `upcoming`, `past`, `cancelled` | Defines event lifecycle state |
| `organization_status` | `active`, `partner`, `inactive` | Defines organization status |
| `registration_status` | `registered`, `waitlisted`, `attended`, `no_show`, `cancelled` | Defines participation state |
| `quarter_code` | `Q1`, `Q2`, `Q3`, `Q4` | Defines fiscal quarter |
| `disclosure_doc_type` | `ordinance`, `resolution`, `executive_order`, `bac_document`, `financial_statement`, `program_outcome`, `other` | Defines document category |
| `doc_state` | `ok`, `partial`, `issue` | Defines quarterly compliance state |
| `submission_state` | `submitted`, `late`, `missing` | Defines monthly submission state |
| `ticket_status` | `received`, `in_progress`, `resolved`, `closed` | Defines citizen ticket progress |
| `service_status` | `operational`, `maintenance`, `notice` | Defines service advisory state |
| `barangay_compliance_status` | `compliant`, `pending`, `overdue` | Defines barangay-level compliance summary |

## Summary

This data dictionary reflects the actual structure of LYDO Connect as a youth governance, transparency, and citizen service platform. It replaces the marketplace-oriented dictionary of the template and gives a more accurate basis for discussion in the methodology chapter.
