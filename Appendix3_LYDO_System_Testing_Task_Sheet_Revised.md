## PART I: SYSTEM TESTING TASK SHEET (USER)

**LYDO Connect: A Progressive Web Application for Information, Participation, and Transparency for Local Youth Development Offices**

Good day!

Thank you for participating in the system testing of LYDO Connect. This platform is designed to help youth and citizens access youth-related information and services, including programs, events, organization information, transparency records, and citizen concern reporting.

What is this task sheet all about?  
This task sheet will guide you through the key user-side features and workflows of LYDO Connect. It includes practical tasks to help evaluate clarity, usability, and functionality during actual use.

How to use this sheet:
1. Complete each task listed below at your own pace.
2. Check the "Completed" box once you finish each task.
3. Write comments, issues, or suggestions in the "Comments / Issues" column.
4. You may explore additional pages or workflows beyond the listed tasks.

| Task | Completed | Comments / Issues (Optional) |
|---|---|---|

| 1. Launch the LYDO Connect web application. |  |  |
| 2. Explore the homepage and main navigation menu. |  |  |
| 3. Check whether the purpose of the platform is clear on first use. |  |  |
| **Task 2: Account Access (Sign Up / Sign In)** |  |  |
| 1. Open the Sign Up and Sign In pages. |  |  |
| 2. Review required fields and input guidance. |  |  |
| 3. If credentials are available, sign in and continue to protected user pages. |  |  |
| **Task 3: Policy Agreement Flow** |  |  |
| 1. After sign in, observe whether Terms/Privacy agreement is shown when required. |  |  |
| 2. Review policy content clarity and readability. |  |  |
| 3. Confirm that accepted users can proceed to protected features. |  |  |
| **Task 4: Programs and Registration** |  |  |
| 1. Open the Programs page and review available program records. |  |  |

| **Task 5: Events and Registration** |  |  |
